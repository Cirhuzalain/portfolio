I. Monolingual

For the past few years, several methods have leveraged the huge amount of data available on the web to acquire monolingual corpus in English [1, 2]. In this work we scraped websites from four major news publishers (RFI, BBC, DW & VOA). We start with a seeding link obtained from the Common Crawl repository containing links dating over a decade (from 2008 to 2020). We then analyzed the in memory representation of these web pages and create key identifiers that can be use to retrieve document summary pairs efficiently. In addition to this, we parallelized the final documents retrieval operation by using the tor network. Furthermore, we removed duplicate, short documents and create a multilingual dataset of around 500 000 documents in English, French and Swahili containing articles professionally written by the main news articles author. Finally, we performed a group by language operation that maps all documents from their respective publisher website in the previously-mentioned languages. II. Cross-lingual

Given the challenging nature of acquiring cross-lingual dataset due to the discrepancy in the source and target languages. Inspired by [3] work, we utilized the forward translation as a method for acquiring cross-lingual target from the previously gathered multilingual dataset. We leveraged existing models (https://huggingface.co/Helsinki-NLP) (models : opus-mt- en-fr, en-swc, fr-en, fr-swc, swc-en, swc-fr)} available in the transformer library [4] as machine translation service to construct a final cross-lingual dataset by relying on the forward translation to acquire the corresponding cross-lingual target for the source document. The monolingual target is translated by the aforementioned models from one language to another (forward translation). The following figure depicts the forward translation operation.

We have used the forward translation operation for most of our datasets. However due to the fact that current machine translation system may behave poorly for some examples, We can extend the previously-mentioned operation with a back translation whereby the CDS reference is translated back to the same language as a monolingual target. The combination of these two operations is known as round-trip translation (RRT). The back translation is quintessential to do a filtering operation by leveraging the ROUGE[5] score between the round-trip translated reference and the monolingual target reference. In our truncated experiment given the context, the result with and without this filtering were on par for the Swahili source document. Further investigation may be needed to ascertain the use of that operation in the complete RRT for the French and English source document.
